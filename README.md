## 🚀 Flagship Projects

### 🤖 AI-Powered Corporate Travel Management System — A2A Orchestration
Agent-to-Agent orchestration engine running 10+ concurrent AI agents, cutting response latency by ~40%.

<details>
<summary><b>Architecture & impact details</b></summary>
<br>

- Designed the A2A orchestration engine using Python `ThreadPoolExecutor`, enabling 10+ agents to run concurrently
- Integrated Amadeus, Google Vision OCR, Google Maps, and OpenWeather APIs for real-time travel data aggregation
- Built fallback and failover mechanisms, sustaining ~99% system availability during upstream API failures
- Implemented Flask + SocketIO backend for real-time, event-driven updates
- Deployed containerized services on GCP Cloud Run, reducing infrastructure overhead by ~30%
- Enabled structured LLM-based decision workflows using Gemini models for automation and response generation
- Also home to **JARVIS**, a real-time voice AI agent built on a Google Gemini Live WebSocket pipeline

</details>

### 📊 Financial RAG System
Retrieval-augmented platform improving search relevance by ~25–30% and cutting manual lookup effort by ~50%.

<details>
<summary><b>Architecture & impact details</b></summary>
<br>

- Built an end-to-end pipeline for document ingestion, embedding generation, and vector-based retrieval using pgvector
- Improved search relevance and response accuracy through semantic chunking and optimized embeddings
- Developed REST APIs with FastAPI for query handling and retrieval workflows
- Built a React frontend for interactive querying and visualization of financial insights

</details>

### 🗂 DSV360 — Project & Workforce Management Platform
Full project and workforce management platform improving task-tracking efficiency by ~35%.

<details>
<summary><b>Architecture & impact details</b></summary>
<br>

- Built frontend components using React, Redux Toolkit, Material UI, and Ant Design
- Implemented a hierarchical sprint structure (Milestones → Epics → Stories → Tasks → Subtasks)
- Built modules for leave management, approval workflows, and geolocation-based attendance tracking
- Integrated Zoho Catalyst DataStore and analytics for reporting and operational insights

</details>

### 🏥 J&J HR Digital Assistant
LLM-powered HR chatbot that cut HR response times by 85%, part of **JAIDA** — Johnson & Johnson's Generative AI initiative.

<details>
<summary><b>Architecture & impact details</b></summary>
<br>

- Built using LLaMA 3, LangChain, and Azure OpenAI
- Extracted and pre-processed HR documents (PDF/HTML) for ingestion into QDrant, improving retrieval precision
- Implemented LLM orchestration, prompt engineering, and conversational memory for multi-turn, context-aware interactions
- Evaluated and fine-tuned models including GPT-4, LLaMA 2/3, and Falcon for optimal performance

</details>

### 📄 Document Summarization & Compliance Retrieval (RAG)
Two enterprise retrieval systems: a summarization pipeline (~80% semantic match accuracy) and a compliance retrieval engine (~87% retrieval accuracy).

<details>
<summary><b>Architecture & impact details</b></summary>
<br>

- **Document Summarization System:** RAG pipeline built with Python, QDrant, AWS S3, and Sentence Transformers; handled document extraction and ingestion with semantic chunking
- **Vector Search Optimization:** advanced QDrant search features — keyword detection, multi-select filters, soft guardrails — reaching ~90% result relevance
- **Healthcare Compliance (HCC) Retrieval Engine:** end-to-end compliance document retrieval using TensorFlow, Python, and QDrant, supporting ingestion of regulatory PDFs

</details>
