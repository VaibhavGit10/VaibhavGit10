## 🚀 Flagship Projects

### 🤖 AI-Powered Corporate Travel Management System — A2A Orchestration
Agent-to-Agent orchestration engine (Python `ThreadPoolExecutor`) running 10+ concurrent AI agents, cutting response latency by ~40%. Integrated Amadeus, Google Vision OCR, Google Maps, and OpenWeather APIs for real-time travel data; Flask + SocketIO power live updates; deployed on GCP Cloud Run with ~99% availability through fallback/failover design.

### 📊 Financial RAG System
End-to-end retrieval pipeline — document ingestion, embeddings, and pgvector-based semantic search — improving retrieval relevance by ~25–30%. FastAPI backend, React frontend, reducing manual data lookup effort by ~50%.

### 🗂 DSV360 — Project & Workforce Management Platform
React / Redux Toolkit / Material UI / Ant Design frontend with a hierarchical sprint model (Milestones → Epics → Stories → Tasks → Subtasks), improving task-tracking efficiency by ~35%. Includes leave management, approval workflows, and geolocation-based attendance, backed by Zoho Catalyst DataStore.

### 🏥 J&J HR Digital Assistant
LLM-powered HR chatbot (LLaMA 3, LangChain, Azure OpenAI) that cut HR response times by 85%, with QDrant-based retrieval over HR documents and multi-turn conversational memory. Built as part of **JAIDA** — Johnson & Johnson's Generative AI initiative.

### 📄 Document Summarization & Compliance Retrieval (RAG)
RAG summarization pipeline (QDrant, AWS S3, Sentence Transformers) reaching ~80% semantic match accuracy, plus a Healthcare Compliance (HCC) retrieval engine (TensorFlow + QDrant) achieving ~87% retrieval accuracy over regulatory documents.
